# Assignment 2 - Big Data

## Project Overview
This project focuses on implementing a collaborative filtering algorithm to provide recommendations based on user behaviors and preferences. The main goal is to demonstrate the effectiveness of various data preprocessing techniques and evaluation metrics in a big data environment.

## Assignment Requirements
1. **Data Collection:** Use appropriate datasets that represent user preferences and behaviors.
2. **Data Preprocessing:** Clean the data, handle missing values, and prepare it for analysis.
3. **Collaborative Filtering:** Implement collaborative filtering techniques, including user-based and item-based approaches.
4. **Evaluation Metrics:** Evaluate the recommendations using precision, recall, F1-score, and RMSE.
5. **Visualizations:** Create visualizations to display the data distribution, model performance, and recommendations.

## Data Preprocessing Steps
- **Loading Data:** Load the dataset from the source.
- **Data Cleaning:** Remove duplicates and irrelevant features.
- **Handling Missing Values:** Use imputation techniques to fill missing entries.
- **Normalization:** Normalize data to bring it on the same scale.

## Collaborative Filtering Implementation
- **User-Based Filtering:** Use similarities between users to provide recommendations.
- **Item-Based Filtering:** Use similarities between items to suggest related items to users.
- **Algorithm Selection:** Choose the appropriate algorithm based on the dataset characteristics.

## Evaluation Metrics
- **Precision:** Measure the proportion of true positive recommendations.
- **Recall:** Measure the proportion of actual positive cases that were identified.
- **F1-score:** Harmonic mean of precision and recall.
- **RMSE:** Root Mean Square Error for prediction accuracy.

## Visualizations
- **Data Distribution:** Histograms and boxplots to show data characteristics.
- **Model Performance:** Graphs comparing evaluation metrics across different algorithms.
- **Recommendations Visualization:** Charts showing top recommendations for users.

## Conclusion
This comprehensive approach leverages big data technologies to analyze and recommend based on user preferences, employing robust preprocessing and evaluation techniques to ensure accuracy and usability of the recommendation system.
