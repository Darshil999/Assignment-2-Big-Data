# README.md

## How to Reproduce Results
1. Clone the repository using `git clone https://github.com/Darshil999/Assignment-2-Big-Data.git`
2. Navigate to the project directory: `cd Assignment-2-Big-Data`
3. Install the required dependencies by running `pip install -r requirements.txt`
4. Run the main script with `python main.py`

## Dependencies
- Python >= 3.8
- pandas
- numpy
- scikit-learn
- matplotlib

## File Structure
- `/data`: Contains datasets used in the analysis
- `/src`: Python scripts for processing data and generating results
- `/notebooks`: Jupyter notebooks for exploratory data analysis
- `requirements.txt`: List of dependencies
- `main.py`: Main script to execute the analysis

## Expected Outputs
- Output files will be generated in the `/results` directory, including charts and exported data files.
